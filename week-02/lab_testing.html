CSET-110 Lab:
Automated Testing

Time to practice writing tests!

Go through the list of functions we wrote for the lab on Simple JS Functions. Using TDD, make a test for each function listed and then write the correct function in order to make the test pass. Each function should have as many tests to test all the possible outputs.

For example, the isEven function should have a test that results in true, a test that results in false, and also a test that deals with a non-numeric input.
